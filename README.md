# Ultimate Experiences — UE Events Website

Australian silent disco event company. Parent of Byron Bay Silent Disco.

## Pages
- `/` — Landing page (van animation + dancers + typewriter hero)
- `/events` — Upcoming events with filter
- `/how-it-works` — RF headset technology explainer
- `/about` — Story, timeline, values
- `/contact` — Contact form → n8n webhook
- `/family-disco` — Ultimate Family Disco event page
- `/admin` — Admin panel (event templates + QR scanner + integrations)
- `/party` — Party van animation

## Stack
- Pure HTML/CSS/JS — no framework
- n8n webhook: https://n8n.flowberry.org/webhook/ue-ticket
- Vercel deployment
